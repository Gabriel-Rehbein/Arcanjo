## ⚠️ Ações Críticas

### Deletar Usuário
⚠️ **IRREVERSÍVEL!** 

Quando você deleta um usuário:
- ❌ Todos os dados são apagados permanentemente
- ❌ Projetos dele são perdidos
- ❌ Diário e tarefas são apagadas
- ❌ Configurações são removidas
- ❌ Perfil é deletado

O sistema pede confirmação **2 vezes** para proteger contra acidentes.

### Limpar Cache
⚠️ **AFETA TODOS OS USUÁRIOS!**

Isso vai:
- Apagar TODOS os dados locais
- Desconectar todos os usuários
- Limpar cache completo
- Redirecionar para login

## 📊 Interpretando Dados

### Estatísticas
- **Usuários**: Total de contas criadas
- **Projetos**: Total de projetos em todas as contas
- **Diários**: Total de entradas de diário
- **Tarefas**: Total de tarefas agendadas

### Detalhes do Usuário
- **ID**: Identificador único
- **Username**: Nome de usuário
- **Projetos**: Quantos projetos tem
- **Entradas**: Quantas entradas de diário tem
- **Status**: Se é admin ou usuário comum
- **Data de Criação**: Quando criou a conta

## 🆘 Troubleshooting

### Não Consigo Acessar Admin Panel
- Verifique se está logado como "admin"
- Verifique se a senha está correta (Admin@2024)
- Limpe o cache do navegador
- Tente em outro navegador

### Usuário Não Aparece na Lista
- Use a busca para procurar
- O usuário pode ter sido deletado
- Verifique se o username está correto

### Não Consigo Deletar Usuário
- Confirme AMBAS as mensagens
- Verifique se é realmente um admin
- Tente novamente
